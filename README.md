# GitHub Pull Request Attachments

Currently, you can attach GitHub Pull Requests to an incident.
Attachments enable you to connect an item from another system to incident.io.
To attach something, you simply need to:
- Make sure you're connected to the provider (e.g., GitHub) on the integration settings page
- Paste a link into an incident channel, like below
